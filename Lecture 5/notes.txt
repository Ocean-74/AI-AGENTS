n8n Automation: AI-Powered Talent Profiler
-------------------------------------------------------------
This repository contains the documentation for my first n8n workflow. The automation takes user input, categorizes it based on profession, generates a custom AI poem, and manages the entire lifecycle within Airtable.
---------
🛠️ Workflow Architecture

The workflow follows a linear logic path with a conditional branch for profession-based rating.

    -Data Ingestion: Receives input (Name, Looks, Profession) via an n8n Form Trigger.
    -Initial Storage: Creates a new record in Airtable.
    -Logic Switch: Evaluates the Profession dropdown value:
    -Video Editor ➔ Assigns star rating.
    -Graphic Designer ➔ Assigns star rating.
    -AI Manager ➔ Assigns star rating.
    -Database Update: Updates the existing Airtable record with the assigned rating.
    -AI Generation: Passes data to an AI Agent powered by Google Gemini 1.5 Flash.
    -Final Sync: Stores the generated poem back into the Airtable record.
